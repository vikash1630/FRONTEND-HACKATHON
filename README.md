# 📊 Admin Analytics Dashboard

A modern admin dashboard built with **React + Vite** for visualizing churn metrics and system analytics.

## 🚀 Features
- Interactive charts (Line, Bar, Area, Scatter, Composed)
- Dark / Light theme support
- Churn analysis & recommendations
- Responsive premium UI
- Real-time API data

## 🧱 Tech Stack
- React (Vite)
- Tailwind CSS
- Recharts
- Axios
- React Router
- Lucide Icons

## ⚙️ Setup

```bash
npm install
npm run dev
